# 003 - Quirky Quiz Questions

## Description
Create a set of quirky, unexpected questions that reveal your true Hogwarts house. These should NOT be obvious "are you brave or smart?" questions - they should be weird, fun, and psychologically sneaky.

## Acceptance Criteria
- [x] Minimum 10 quirky questions (12 questions created)
- [x] Each answer maps to house points (can give points to multiple houses)
- [x] Questions should be unexpected and fun
- [x] Mix of scenarios, preferences, and hypotheticals

## Example Question Types
- "You find a mysterious door. What do you do?" 
- "Pick a sandwich filling"
- "Your friend is being annoying. Your move?"
- "It's 3am. Why are you awake?"

## House Mapping Logic
Each answer gives weighted points to houses:
- Gryffindor: Bravery, impulsiveness, moral courage, showing off
- Slytherin: Ambition, cunning, self-preservation, resourcefulness  
- Ravenclaw: Curiosity, creativity, wit, overthinking
- Hufflepuff: Loyalty, fairness, patience, snacks
