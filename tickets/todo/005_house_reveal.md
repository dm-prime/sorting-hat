# 005 - Dramatic House Reveal

## Description
Create an epic, theatrical house reveal experience. The Sorting Hat should "think" dramatically before announcing the house.

## Acceptance Criteria
- [x] Suspenseful delay before reveal (2-3 seconds)
- [x] Hat "muttering" animation with quirky phrases
- [x] House name announced with fanfare
- [x] House crest/colors displayed
- [x] Personalized quirky message about why you belong there
- [x] Option to retake the quiz
